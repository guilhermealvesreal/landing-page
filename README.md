# Landing Page

Landing page estática e responsiva desenvolvida para prática de estruturação de layouts e estilização com HTML e CSS.

## Descrição
Este projeto tem como objetivo a criação de uma landing page simples, simulando a apresentação de um produto ou serviço. O foco está na organização do código, clareza visual e aplicação de boas práticas iniciais de front-end.

## Funcionalidades
- Estrutura de layout clara e organizada
- Seções bem definidas para apresentação de conteúdo
- Design responsivo para diferentes tamanhos de tela
- Código legível e de fácil manutenção

## Tecnologias utilizadas
- HTML5
- CSS3

## Como visualizar o projeto
1. Faça o clone do repositório ou o download dos arquivos  
2. Abra o arquivo `index.html` em um navegador web
